  • 第三天：蜈支洲岛玩海一天
    • 蜈支洲岛
    • 住宿：180°海景精选酒店（帆船港/大卫传奇/维景/海上时光/海虹）、三亚明申高尔夫度假酒店(Sanya Mingshen Jinjiang Golf Resort)、三亚天通建国酒店(Sanya Timton International Hotel)、三亚半山锦江海景度假酒店(Sanya Yuhuayuan Hotel)
  • 第四天：酒店 - 南山大小洞天、南山大小洞天 - 天涯海角
    • 5A级 大小洞天景区
    • 4A级 天涯海角

5.海南三亚4日3晚跟团游(5钻)（赠送玩水礼包：明细：“地中海”潜水 、摩托艇、海上自行车、海上独木舟、激情冲浪板、滩游戏（桃花朵朵开+按摩派对）、海鲜火锅。就等你来high！）—>>> 下文将使用F来代替称呼这个旅游方案

行程

  • 第一天：深圳出发前往酒店。三亚君锦滨海酒店 或 三亚国际饭店 或 三亚玉海国际度假酒店
  • 第二天：爱立方，槟榔谷
    • 亚龙湾爱立方
    • 槟榔谷
    • 住宿：三亚君锦滨海酒店 或 三亚国际饭店 或 三亚玉海国际度假酒店
  • 第三天：嗨翻蜈支洲岛一整天~(携带泳衣、拖鞋、墨镜、遮阳帽、防晒霜等装备设备。中餐需要自理，带零食)
    • 蜈支洲岛
    • 住宿：三亚君锦滨海酒店 或 三亚国际饭店 或 三亚玉海国际度假酒店
  • 第四天：天涯海角，大小洞天返回温馨的家
    • 5A级 大小洞天景区
    • 4A级 天涯海角
    • 建议选择20：00以后航班，如航班时间较早无法用晚餐，导游现退餐标。导游会安排送机场。

价格

  • 2686

总结

第一个景区

  • 第一个游玩时间短，游玩景区也相对没有其他那么有意思。
  • 价格较便宜，但是也没有便宜很多。

第二个景区

  • 游玩时间长，游玩景区活动也最多，有篝火晚会，抓螃蟹，热带植物园，活动丰富有意思。
  • 价格较贵，但是如果能接受5号出发（2749），那就完美。

第三个景区

  • 游玩时间长，游玩景区活动多，但是大部分是偏向于观赏类，较适合小清新和装xxx。
  • 价格便宜。

优先级排序

  • 按有趣程度排行： B > C > A
  • 按消费 排行： A > C > B
  • 综合 排行： 周五的B > C > B > A
